# AI-Local 🧠 — Chat tiếng Việt

Mô hình ngôn ngữ train từ đầu theo phong cách nanoGPT, chạy hoàn toàn miễn phí.

Model nhỏ (1.8M tham số) học char-level tiếng Việt, qua 2 giai đoạn:
pre-training + SFT (instruction tuning), có token EOS để biết dừng.

Mã nguồn: https://github.com/luibobo932/AI-local
